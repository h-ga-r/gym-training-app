# 💪 Gym Training App

自分専用のジムトレーニング管理Webアプリです。インターバルタイマーとトレーニング記録に特化し、スマートフォン(特にiPhone)での使用を前提に作られています。

🔗 **Live Demo**: https://161-33-135-200.sslip.io:8443/

## 制作の背景・目的

学費のためジムを一時中断していますが、来年の再開に向けて今から使えるツールを自作したいと考え、このアプリを作りました。既存のトレーニングアプリではなく、自分自身の使い方に完全に合わせた仕様で設計しています。

## 主な機能

### ⏱ インターバルタイマー
- セット間の休憩時間を管理
- 60秒 / 90秒 / 120秒 / 180秒のプリセットボタン + ±10秒の微調整ボタン
- 休憩終了時は音を鳴らさず、画面の点滅で通知(ジムで周囲の迷惑にならないよう配慮)
- タブや他アプリに切り替えても、正確な経過時間を計算して復帰時に反映

### 📋 トレーニング記録
- 部位(胸・背中・脚・肩・腕・腹・有酸素・その他)ごとに種目を管理
- セットごとに記録を残していく方式(1セット記録するたびに、その日・その種目の過去のセット記録も一覧表示)
- 回数はドロップダウン形式で選択(3, 5, 6, 9, 10, 12, 15, 18, 20, 限界まで)
- カレンダー画面から日付を選んで記録。カレンダー上にはその日にトレーニングした部位が一目で分かるように表示される
- 今日の日付のみ記録可能、過去の日付は閲覧専用

## 使用技術

- **フロントエンド**: HTML / CSS / JavaScript(フレームワークなし、素のJSのみ)
- **データ保存**: ブラウザのlocalStorage(バックエンドなし)
- **デプロイ**: Oracle Cloud Infrastructure(Always Free枠のVM) + Nginx
- **HTTPS化**: Let's Encrypt(Certbot) + sslip.io(独自ドメイン不要でIPアドレスからHTTPS対応)

## 今後追加予定の機能

- 目標体型画像・参考スペックをもとにした目標値の算出
- 体重・体脂肪率をもとにしたタンパク質・脂質・カロリー目標の自動計算

## 既知の制約

- データはブラウザのlocalStorageに保存されるため、別端末・別ブラウザとは同期されません
- iOS Safariの仕様上、バイブレーションによる通知には対応していません(画面点滅で代替)
- タイマーの通知は画面をアプリに戻したタイミングで反映されます(他アプリ使用中はリアルタイム通知されません)
